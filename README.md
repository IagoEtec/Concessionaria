# Select Car Motors - Sistema de Concessionária

Sistema web para concessionária de veículos esportivos e exclusivos.

## Funcionalidades

- Cadastro e login de usuários
- Catálogo de carros disponíveis
- Área administrativa para gerenciar veículos
- Design responsivo e moderno

## Tecnologias

- PHP
- MySQL
- HTML5
- CSS3

## Instalação

1. Importe o arquivo `conce.sql` no seu MySQL
2. Configure as credenciais do banco em `db.php`
3. Coloque os arquivos em um servidor web com PHP habilitado

## Estrutura do Banco

- `login`: Credenciais de acesso
- `perfil_user`: Perfil dos usuários
- `carros`: Catálogo de veículos
